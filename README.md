# Healthcare Utilization Analysis

This project analyzes patient visit patterns, diagnosis frequency, and overall healthcare utilization using Python, SQL, and Pandas. The goal is to uncover meaningful trends that support operational decision-making, improve patient care, and identify high-utilization patients who may benefit from additional support.

## Tools Used
- Python
- Pandas
- SQL (SQLite)
- Jupyter Notebook

## Key Steps
- Loaded and cleaned patient, visit, and diagnosis datasets
- Joined multi-table healthcare data using SQL
- Created aggregated metrics (visits per patient, monthly trends, diagnosis frequency, visit duration)
- Performed exploratory analysis to identify utilization patterns and high-need patients
- Summarized findings, insights, and recommendations in a structured notebook

## Files
- `Analysis.ipynb` – main analysis notebook

## Purpose
This project was completed as part of a data analytics learning journey and is intended to demonstrate skills in data cleaning, analysis, and interpretation.
